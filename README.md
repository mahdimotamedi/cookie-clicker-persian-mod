# Cookie Clicker Persian Translation

A complete Persian language mod for the Steam version of Cookie Clicker.

## Features

- Full Persian translation
- Complete news ticker translation
- Right-to-left interface support
- Translated upgrades, achievements, and minigames
- Compatible with Steam achievements

## Installation

Install from steam workshop:
https://steamcommunity.com/sharedfiles/filedetails/?id=3768204281

OR

1. Download the latest release.
2. Extract the mod folder into:

   `Cookie Clicker/resources/app/mods/local/`

3. Open the game and go to **Options → Manage Mods**.
4. Enable the mod and restart the game.
5. Select Persian from the language menu.
